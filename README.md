# Synchronous-FIFO
Designed and implemented a parameterized synchronous FIFO using Verilog HDL. The FIFO supports configurable data width and depth, circular buffer architecture, and Full/Empty status flag generation.

It features:
- Parameterized FIFO depth and width
- Circular read/write pointers
- Full and Empty flag generation
- Synchronous read and write operations
- Count-based occupancy tracking
- Functional verification using a Verilog testbench

Tools Used:
- Verilog HDL
- EDA Playground
- Icarus Verilog

Files:
FIFO.v -> FIFO RTL design
FIFO_tb.v -> Testbench
Output.png -> Simulation Waveforms
